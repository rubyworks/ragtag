== OVERVIEW

RagTag if a Ruby-based template attribute lanaguage (TAL). Yea, it's an
acronym. Don't blame me, blame those Zope people ;)

RagTag is only loosely based on the Zope TAL sepcification.
It's first priority has been to provide a good attribute based
template system for _Ruby_.


== USAGE

For the moment please see the WIKI[http://wiki.github.com/rubyworks/ragtag/].
